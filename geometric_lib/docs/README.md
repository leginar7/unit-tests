
# Руководство по модулям: `rectangle.py`, `square.py`, `triangle.py`, `circle.py`

## Краткое описание
В проекте собраны простые функции для вычисления площади и периметра (или длины окружности) у основных геометрических фигур:

- прямоугольника,
- квадрата,
- треугольника,
- круга.

Функции не требуют сторонних библиотек и возвращают числовые результаты на основе введённых параметров. Такой набор может пригодиться при обучении, создании тестовых калькуляторов или в задачах, связанных с элементарной геометрией.

---

## Модуль `rectangle.py`

Функции для работы с прямоугольниками.

### `area(a, b)`
Возвращает площадь прямоугольника.  
- **Аргументы:**  
  - `a` — ширина;  
  - `b` — высота.  
- **Результат:** число (площадь).

Пример:
```python
from rectangle import area

print(area(5, 10))  # 50
```

### `perimeter(a, b)`
Возвращает периметр прямоугольника.  
- **Аргументы:**  
  - `a` — ширина;  
  - `b` — высота.  
- **Результат:** число (периметр).

Пример:
```python
from rectangle import perimeter

print(perimeter(5, 10))  # 30
```

## Модуль `square.py`

Функции для квадратов.

### `area(a)`
Считает площадь квадрата по его стороне.  
- **Аргументы:** `a` — длина стороны.  
- **Результат:** число (площадь).

Пример:
```python
from square import area

print(area(6))  # 36
```

### `perimeter(a)`
Считает периметр квадрата.  
- **Аргументы:** `a` — длина стороны.  
- **Результат:** число (периметр).

Пример:
```python
from square import perimeter

print(perimeter(6))  # 24
```

---

## Модуль `triangle.py`

Функции для треугольников.

### `area(a, h)`
Считает площадь по основанию и высоте.  
- **Аргументы:**  
  - `a` — основание треугольника;  
  - `h` — высота, проведенная к основанию треугольника.  
- **Результат:** число (площадь).

Пример:
```python
from triangle import area

print(area(10, 6))  # 30
```

### `perimeter(a, b, c)`
Считает периметр по трём сторонам.  
- **Аргументы:** `a`, `b`, `c` — стороны треугольника.  
- **Результат:** число (периметр).

Пример:
```python
from triangle import perimeter

print(perimeter(3, 4, 5))  # 12
```

---

## Модуль `circle.py`

Функции для окружностей и кругов.

### `area(r)`
Вычисляет площадь круга.  
- **Аргументы:** `r` — радиус.  
- **Результат:** число с плавающей точкой (площадь).

Пример:
```python
from circle import area

print(area(5))  # 78.53981633974483
```

### `perimeter(r)`
Вычисляет длину окружности.  
- **Аргументы:** `r` — радиус.  
- **Результат:** число с плавающей точкой (длина окружности).

Пример:
```python
from circle import perimeter

print(perimeter(5))  # 31.41592653589793
```
